### Exercise 1.1: Repository Settings Navigation
**Objective**: Navigate to and understand the structure of GitHub repository settings.

**Tasks**:
1. Open a GitHub repository and locate the "Settings" tab in the repository navigation
2. Understand that settings control repository behavior, access permissions, and feature enablement
3. Document the distinction between repository settings and profile settings
4. Identify that settings are organized into logical sections: General, Access, Code & Automation, Security, Integrations

**Deliverable**: Repository settings navigation guide with structural overview and organizational sections.

---

### Exercise 1.2: General Settings Configuration
**Objective**: Configure basic repository properties and feature management.

**Tasks**:
1. Access the General settings section and explore available configuration options
2. Practice changing repository name and description settings
3. Understand default branch configuration and branch switching options
4. Document feature toggles: enable/disable issues, wikis, discussions, sponsorships, projects
5. Configure social preview images for repository sharing

**Deliverable**: General settings configuration guide with feature management options documented.

---

### Exercise 1.3: Danger Zone Management
**Objective**: Understand and document critical repository management operations.

**Tasks**:
1. Locate the "Danger Zone" section at the bottom of General settings
2. Document repository visibility controls: public to private conversion
3. Understand branch protection rule management and repository ownership transfer
4. Learn about repository archival options for inactive projects
5. Understand permanent deletion implications and warnings

**Commands** (conceptual):
```
Repository Visibility: Public ↔ Private
Branch Protection: Enable/Disable rules
Ownership Transfer: To another account
Archive: For inactive repositories
Delete: Permanent removal (irreversible)
```

**Deliverable**: Danger zone operations guide with risk assessment and safety considerations.
