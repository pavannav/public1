### Exercise 1.1: GitHub Homepage Navigation
**Objective**: Familiarize yourself with the GitHub homepage layout and elements

**Tasks**:
1. Log into your GitHub account and document the homepage elements:
   - Main navigation bar items
   - Search functionality location
   - Profile menu options
   - Plus/Create menu options
2. Identify and document recommended sections:
   - "Getting started" suggestions
   - Learning resources
   - Tool recommendations (VS Code, GitHub Desktop)
3. Create a visual map of the homepage layout

**Deliverable**: Homepage element inventory with visual layout documentation

---

### Exercise 1.2: Navigation Bar Exploration
**Objective**: Master the top navigation bar functionality

**Tasks**:
1. Document each navigation bar element and its purpose:
   - Search bar capabilities
   - GitHub Copilot icon location and purpose
   - Plus (+) menu options and submenus
   - Issues, Pull requests, and Discussions tabs
   - Profile dropdown menu items
2. Test the search functionality with different queries
3. Explore the hamburger menu and compare it to the top navigation
