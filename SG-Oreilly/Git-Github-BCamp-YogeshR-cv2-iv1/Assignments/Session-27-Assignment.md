### Exercise 1.1: Understanding Local vs Remote Repositories
**Objective**: Compare the limitations of local-only repositories with remote repositories

**Tasks**:
1. Create a local Git repository with several commits
2. Document what happens if you "lose" access to this repository (simulated)
3. List at least 5 limitations of working with local repositories only
4. Explain how each limitation is solved by remote repositories

**Deliverable**: Written comparison document highlighting local vs remote repository limitations and benefits

---

### Exercise 1.2: Collaboration Scenarios
**Objective**: Understand remote repository collaboration benefits

**Tasks**:
1. Create a scenario document describing a 3-developer team working on a web application
2. Detail what challenges they would face using only local repositories
3. Explain how using a remote repository solves each challenge
4. Include specific examples of workflow improvements

**Deliverable**: Team collaboration scenario document with problem-solution pairs

### Exercise 1.3: Backup and Recovery Planning
**Objective**: Appreciate the backup benefits of remote repositories

**Tasks**:
1. Create a risk assessment document for local repository loss scenarios:
   - Hardware failure (laptop crash)
   - Accidental deletion
   - Data corruption
   - Theft/loss of device
2. For each scenario, explain:
   - Impact without remote backup
   - Recovery process with remote repository
   - Prevention strategies

**Deliverable**: Risk assessment matrix with recovery procedures
