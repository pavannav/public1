### Exercise 1.1: GitHub Account Registration Process
**Objective**: Document the complete GitHub account creation workflow

**Tasks**:
1. Visit github.com and document the homepage elements
2. Identify and document all sign-up options available
3. Create a step-by-step checklist for account registration
4. Note any prerequisites or information needed before starting

**Deliverable**: Step-by-step registration checklist with homepage element inventory

### Exercise 1.2: Account Creation Methods Comparison
**Objective**: Understand different ways to create a GitHub account

**Tasks**:
1. Compare the two account creation methods:
   - Google account sign-up
   - Email-based registration
2. Document pros and cons of each method
3. Note any differences in the registration process
4. Create decision guidelines for choosing a method

**Deliverable**: Method comparison document with recommendation guidelines

---

### Exercise 1.3: Username Selection Strategy
**Objective**: Develop best practices for choosing a GitHub username

**Tasks**:
1. Research GitHub username requirements and limitations
2. Create a list of 5 potential usernames following best practices
3. Document criteria for professional usernames
4. Explain the importance of username choice for professional presence

**Requirements to research**:
- Character limitations
- Reserved words
- Availability checking
- Professional considerations

**Deliverable**: Username selection strategy document with criteria and examples
