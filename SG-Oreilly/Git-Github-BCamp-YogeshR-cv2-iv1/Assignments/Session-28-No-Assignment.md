# Session 28: No Assignment

This session contains introductory conceptual content about GitHub without any practical exercises or demonstrations that would warrant assignments.

Content covered:
- Git vs GitHub distinction (tool vs service)
- GitHub as a cloud-based platform for hosting Git repositories
- Collaboration features (pull requests, issue tracking, project boards)
- GitHub's role in open source development
- Alternative platforms (GitLab, Bitbucket, Azure Repos)
- GitHub's community and ecosystem advantages

Practical assignments for GitHub account setup, repository creation, and hands-on usage will be available in subsequent demonstration sessions.
