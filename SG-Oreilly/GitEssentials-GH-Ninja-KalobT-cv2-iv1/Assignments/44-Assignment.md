## Exercise 2.1: Troubleshooting GitHub Pages Issues
**Objective**: Learn to diagnose and resolve common GitHub Pages deployment problems

**Tasks**:
1. Document scenarios where GitHub Pages shows 404 errors
2. Identify common causes: incorrect repository naming, deployment delays
3. Practice troubleshooting steps for GitHub Pages issues

**Common Issues to Document**:
- Repository naming typos (must be exactly [username].github.io)
- Deployment timing delays (may take several minutes)
- Private repository access issues (must be public)
- Branch configuration problems

**Deliverable**: Comprehensive troubleshooting guide covering common GitHub Pages issues and their solutions

## Exercise 2.2: Repository Naming Verification
**Objective**: Master the critical requirement for exact repository naming conventions

**Tasks**:
1. Verify your repository name matches exactly: [username].github.io
2. Understand the importance of case sensitivity and exact spelling
3. Practice checking and correcting repository names if needed

**Commands/Steps**:
```
Repository Name Verification Checklist:
□ Repository name exactly matches: [your-username].github.io
□ No typos or extra characters
□ Uses .github.io not .github.com
□ All lowercase (GitHub usernames are case-insensitive but convention is lowercase)
□ Located in your personal account, not an organization
```

**Deliverable**: Verified repository naming with documentation of exact name matching requirements

## Exercise 2.3: Deployment Status Monitoring
**Objective**: Learn to monitor and interpret GitHub Pages deployment status indicators

**Tasks**:
1. Identify different GitHub Pages deployment status messages
2. Understand the "Your site is published at" green success indicator
3. Document timing expectations for initial and subsequent deployments

**Status Indicators to Document**:
- "Your site is published at [URL]" - Successful deployment
- "Your GitHub Pages site is currently being built from the master branch" - In progress
- 404 error page - Configuration or deployment issue
- "GitHub Pages is disabled" - Pages not configured

**Deliverable**: Status monitoring guide with interpretation of different deployment indicators and expected timing
