## Course Overview

Welcome to "Linux Networking Basics and Beyond," a comprehensive video course designed to provide the expertise needed to configure networking on various Linux server distributions. The course is taught by Dave Prowse, an IT professional with over 20 years of experience.

This course focuses on Linux when used as a server, recognizing that the great majority of servers on the cloud and in today's networks run Linux in one form or another. The ability to network these servers together—whether physical or virtual—is essential for modern IT infrastructure.

## Course Structure and Learning Path

The course employs a **building block approach** designed to develop knowledge in an orderly way. Students are advised to cover lessons in order without skipping ahead, as concepts build upon each other with frequent references to previous material.

**Course Statistics:**
- **Total Lessons**: 17 lessons
- **Total Modules**: 4 modules
- **Lesson Format**: Short, manageable sub-lessons

## Course Modules Breakdown

### Module 1: Set-up and Networking 101
- Initial system setup with recommendations
- Working with terminal and text editors
- Computer networking basics
- TCP/IP fundamentals
- OSI model concepts
- Additional foundational networking concepts

### Module 2: Networking Services
Core focus on actual networking service implementations across distributions:
- **Debian and Server**: Traditional networking service configuration
- **Ubuntu**: NetworkD and netplan configuration
- **Fedora, Red Hat Enterprise Linux, CentOS**: NetworkManager configuration

### Module 3: Commands and Configurations
Extensive coverage of essential networking commands and tools:
- IP command operations
- Network connectivity testing (ping)
- DNS lookup utilities (dig)
- NetworkManager command-line interface (NMCLI)
- Hostnames configuration
- DNS setup and management
- Additional configuration commands

### Module 4: Connecting Between Hosts with the Command Line
Network connectivity and file transfer tools:
- SSH (Secure Shell) for remote access
- SFTP for secure file transfer
- Rsync for efficient file synchronization
- SMB for Windows-compatible file sharing
- Additional networking tools (wget, curl, etc.)
