## Questions and Answers

### 1. What is cloud computing?
**Answer:** Cloud computing is the delivery of computing services—like servers, storage, databases, networking, software, and AI—over the internet instead of using physical hardware or local servers.

**Note:** Cloud is a conceptual model; platforms such as AWS, Azure, and Google Cloud provide practical implementations of this model.

### 2. What is AWS?
**Answer:** AWS (Amazon Web Services) is a cloud computing platform provided by Amazon. It offers a wide range of services for building, deploying, and managing applications and infrastructure over the internet.

### 3. What is the difference between cloud and AWS?
**Answer:** Cloud is the overarching concept of delivering IT resources over the internet; AWS is one specific vendor/platform that implements cloud-computing services (along with Azure, Google Cloud, etc.).

### 4. What are the five advantages of using AWS over on-premise servers?
**Answer:**

1. **Cost-saving** – No upfront hardware purchases; pay-as-you-go pricing.
2. **Scalability & Elasticity** – Instantly scale resources up or down based on demand.
3. **High Availability & Reliability** – Multi-AZ, multi-region infrastructure with built-in redundancy.
4. **Security & Compliance** – Enterprise-grade security controls, compliance certifications, DDoS protection.
5. **Speed & Agility** – Provision resources in minutes, iterate fast, focus on innovation instead of maintenance.
