# 19.6 How DB Executes CTAS

So now let's see how the database server can execute the CTAs.So you may ask the data analyst.You are writing a select into sales.So that means you are creating a table from a query.So once you executed the database engine to identify the query first and then execute it.So it's going to go and select data maybe from the table orders and retrieve the data and return theresults.Now once the database engine has the results it's going to go and do two things based on the results.It's going to create.First a metadata in the system catalog for the table called sales.And then it's going to go and create a table called sales in the users data.And this sales table can hold the data from the result of the query.So it is like any other database table.It can stores the metadata at the system catalog.And it can store the data in the disk under the users data.
