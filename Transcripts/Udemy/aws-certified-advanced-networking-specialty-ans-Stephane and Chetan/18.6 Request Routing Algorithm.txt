So it will...the idea here is that you really want to balance,based on how busy an EC2 instance is going to be.And we assume that an EC2 instance is not busy,if it has a low number of pending or unfinished requests.Okay. Now this mode works forthe ALB and the CLB mode HTTP or HTTPS.Okay.
